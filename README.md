# CIU_prototipo_video
 

```
Proponer un concepto y su prototipo de combinación de salida gráfica en respuesta a
una captura de vídeo. Una sugerencia puede ser inspirarse en trabajos ya existentes, como
los de la galería.
La entrega se debe realizar a través del campus virtual, remitiendo una memoria o el
enlace a un screencast que además de describir las decisiones adoptadas, identifique al remitente,
las referencias y herramientas utilizadas, cuidando el formato y estilo. Como material
adicional debe incluirse el enlace al código desarrollado (p.e. github), con su correspondiente
README, y un gif animado, de almenos 5 segundos, que ilustre ejecución del prototipo.


```
